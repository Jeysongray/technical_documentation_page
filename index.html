Introduction

Web programming, also known as web development, is the creation of dynamic web applications. Examples of web applications are social networking sites like Facebook or e-commerce sites like Amazon.

There are two broad divisions of web development – front-end development (also called client-side development) and back-end development (also called server-side development).

Prerequisite

This tutirial assumes you have zero experience coding. However, to follow the steps provided here with ease, you need basic knowledge of how the World Wide Web works and computer usage. This means you should be familiar with keyboard shortcuts such as:

Ctrl+s Ctrl+z Ctrl+v, etc.

Should you encounter any difficulties, apply your technical sophistication by simply 'Googling' the problem.

Web Tiers

Front-end development refers to constructing what a user sees when they load a web application – the content, design and how you interact with it. This is done with three codes – HTML, CSS and JavaScript.

After this lesson, you should be able to write sipmple HTML and CSS code like creating a HTML document and styling it: h1 {
background-color: red;
color: blue;
}

This changes the text color of the h1 element to blue and gives it a background color of red. h1 {
font-weight: bold;
}

This makes the font bold.

HTML, short for Hyper Text Markup Language, is a special code for ‘marking up’ text in order to turn it into a web page. Every web page on the net is written in HTML, and it will form the backbone of any web application. CSS, short for Cascading Style Sheets, is a code for setting style rules for the appearance of web pages. CSS handles the cosmetic side of the web. Finally, JavaScript is a scripting language that’s widely used to add functionality and interactivity to web pages. In HTML and CSS, the code for creating and styling document are known as tags or elements and selectors respectively. Some examples below:

  • Tables
  • Forms
  • Buttons
  • CSS grid
  • Paragraphs
Text Editors

Choosing a text editor also known as Integrated Development Environment is pretty easy and there's a whole lot to choose from: Sublime, Bracket, Visual Studio Code, Atom, Notepad, Notepad++ and many more.

As a beginner, I would reccommend something simple like Notepad if you're using a windows OS or Notepad++. After a while, you may try some of the others and may come to find a preference. I especially like VS code as I have worked on all the IDEs mentioned above and just enjoy the VScode interface. But choosing an IDE should be the least of your worries at this point.

Hello World

Writing your first HTML code is anything but complicated. In <HTML5>, which is the latest HTML version, you begin by declaring a <!Doctype> like so:

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport"content="width=device-width, initial-scale=1.0">
<meta http-equiv="X-UA-Compatible" content="ie=edge">
<title>Hello World</title>
</head>
<body>
...content of your web page goes here.
</body>

The above code is the standard declaration. You don't need to memorize this as a good number of text editors declares this for you. For instance, in VScode, type:

Ctrl + ! + return and the entire doctype code above would be automatically displayed! Like I said, easy right?
